- 브랜치 설정

git Fork 후

git clone

git remote add upstream 중앙주소(클론할때 나온 주소)



- 브랜치 생성

git branch feature

+

git checkout feature

git checkout -b feature/login



- 지우기

git checkout -d feature



- 명상하기

git push origin feature/login



git checkout master

git pull upstream
